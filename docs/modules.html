  • Defined in shared/types.ts:27

Type declaration

  • configuration: ConfigurationGET | ConfigurationPOST

    Each Provider button will use this configuration for its button. The default value is {"method": "GET"}

  • corners: CornerType

    The type of corner for the button: rounded or sharp. The default value is "rounded"

  • Optional extraProviders?: ProviderRecord[]

    Used for testing. Do not use in production

  • fluid: boolean

    This controls the width of the button: when fluid it will fill all the available space. The default value is "false".

  • lang: Languages

    The language used for the UI. The default value is "it".

  • Optional mapping?: Record<string, string>

    An object containing the mapping for the providers. This is useful when a Service Provider identifies the IDP with a different string than the entityID

  • Optional onProviderClicked?: (providerEntry: ProviderRecord) => void

    This is called when a user clicks on a provider button.

    param

    The full entry of the provider clicked

      • (providerEntry: ProviderRecord): void

      • Parameters

        • providerEntry: ProviderRecord

        Returns void

  • Optional onProvidersHidden?: () => void

    This is called when the providers are hidden on the screen (fully hidden)

      • (): void

      • Returns void

  • Optional onProvidersShown?: () => void

    This is called when the providers are shown on the screen (fully loaded)

      • (): void

      • Returns void

  • protocol: Protocols

    The protocol to use for the current instance. Only Providers who support the declared protocol are enabled. The default value is "SAML".

  • size: Sizes

    The size of the button. Options are: "sm" (small), "md" (medium) and "l" (large). The default value is "md".

  • supported: ProviderRecord["entityID"][]

    The list of entityID supported by the button instance. The default value is [] (none).

  • theme: ColorTheme

    The theme used for the button:

    • "positive" has a blue background with white text,
    • "negative" has a white background and blue text. The default value is "positive"
  • url: string

    The URL used by the buttons. It can be either absolute or relative. It must contains the "{{idp}}" string in it, which will be replaced by the entityID of each provider (unless specified otherwise with the mapping prop - see below). This props is mandatory.
